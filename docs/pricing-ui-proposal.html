실링스티커 / 추가상품 상품 카드 가격 표시 방안

작성일: 2026.03.13 | 더기프트 퍼스트몰
1 현황 및 문제점
현재 표시 <strong class="price-real">5,900원</strong> → 단품가만 노출
문제점 1 단품가가 이미 할인된 가격인데, 정가 대비 할인 폭을 고객이 인지할 수 없음
문제점 2 청첩장과 함께 구매 시 세트가 할인 혜택이 상품 목록에서 보이지 않음
문제점 3 단품가 / 세트가 가격 차이를 통한 세트 구매 유도가 불가능
2 가격 체계 구조
정가 (소비자가) 기준이 되는 원래 가격. 취소선으로 표시하여 할인 기준점 역할
세트가 청첩장과 함께 구매 시 적용. 정가 대비 높은 할인율 → 세트 구매 유도
단품가 단독 구매 시 적용. 정가 대비 기본 할인율 적용
💡 핵심 전략: 정가를 기준선으로 노출하여 세트가/단품가 모두 "할인 중"이라는 인식을 만들고, 세트가의 더 높은 할인율로 청첩장 동시 구매를 유도합니다.

예시 — 블렌딩 실링스티커 로즈 쿼츠:

정가 8,000원
세트가 45% OFF → 4,400원 (청첩장 함께 구매 시)
단품가 26% OFF → 5,900원 (단독 구매 시)
3 UI 표시 방안 (권장안)

✅ A안: 두 가격 동시 표시 (권장)

  • 카드 내 가격 영역에 세트가와 단품가를 한눈에 표시
  • 각각 배지 라벨(세트 / 단품)로 구분
  • 정가는 취소선으로 표시, 할인율을 강조
  • 장점: 한 화면에서 두 가격을 비교 가능, 세트가의 혜택이 즉시 체감됨
  • 단점: 카드 높이가 다소 증가

B안: 탭 전환 방식

  • 카드 상단에 "세트구매 / 단독구매" 탭을 두고 전환
  • 장점: 카드 높이 유지, 깔끔한 레이아웃
  • 단점: 한 번에 비교 불가, 탭 인터랙션 필요

C안: 세트가 강조 + 단품가 보조 표시

  • 세트가를 메인 가격으로 크게 표시, 단품가는 작은 텍스트로 보조
  • 장점: 세트 구매 유도 효과 극대화
  • 단점: 단독 구매 고객에게 혼란 가능성

AS-IS vs TO-BE 비교

현행 단품가 단일 표시 → 개선안 정가/세트가/단품가 동시 표시

현행 AS-IS: 단품가만 표시
→
개선 TO-BE: 3가지 가격 표시
AS-IS 문제점
  • 단품가만 노출, 할인 여부를 알 수 없음
  • 세트가 혜택이 목록에서 보이지 않음
  • 정가 기준이 없어 가격 매력도 전달 불가
  • 세트 구매 유도 장치 부재
TO-BE 개선 효과
  • 정가 취소선으로 할인 중임을 즉시 인지
  • 세트가/단품가 동시 노출로 비교 가능
  • 세트가 할인율 강조로 동시 구매 유도
  • 할인 뱃지로 시각적 주목도 향상

5 기대 효과
  • 🎯 세트 구매 전환율 증가: 세트가의 높은 할인율이 시각적으로 즉시 인지되어 청첩장 동시 구매 유도
